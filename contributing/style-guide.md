# Style Guide

Detailed style guidelines for contributing to UCLCHEM.

## Python Style

*Detailed guidelines coming soon*

### Code Formatting

Use Black formatter:

```bash
black src/uclchem/
```

### Naming Conventions

- Functions: `snake_case`
- Variables: `snake_case`
- Constants: `UPPER_SNAKE_CASE`
- Classes: `PascalCase`

### Documentation

NumPy-style docstrings for all public functions.
