UCL Chemical Codes

A repository hub for UCL associated chemical and PDR codes.

UCLCHEM 3D-PDR UCL PDR

There are three codes available on this site. UCLCHEM is a gas-grain time-dependent chemical code. 3D-PDR is the first fully 3D code for PDR regions and a 1D version - UCL_PDR - will also be available on this site. Links for each code are above and the individual pages for each code have full descriptions and links to the code repositories and manuals.
