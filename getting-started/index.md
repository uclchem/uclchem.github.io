# Getting Started

Welcome to UCLCHEM! This guide will help you get up and running with UCLCHEM in just a few minutes.

## Prerequisites

Before installing UCLCHEM, make sure you have:

- **Python 3.8 or later**
- **A Fortran compiler** (gfortran recommended)
- **pip** package manager
